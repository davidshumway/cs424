Overview

The spatially-based dataset consists of nearly 30,000 power plants spread out across the globe.

Data types in the original dataset

latitude: Numeric
longitude: Numeric
country_long: String
capacity_mw: Numeric
primary_fuel: String
estimated_generation_gwh: Numeric

Data issues

There were no significant issues relating to data quality when working with the dataset.
